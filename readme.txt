== Description ==

WooCommerce plugin for Chinavasion dropshipping. Import Chinavasion products, check Chinavasion stock status, send orders to Chinavasion. Requires WooCommerce and a Chinavasion API key.

Major features include:

* Import products from Chinavasion API.
* Check Chinavasion stock status.
* Send orders to Chinavasion.

== Installation ==

Install the plugin normally.
